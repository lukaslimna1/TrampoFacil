## 📜 Manifesto & Filosofia
O **Trampo Fácil** nasceu para destruir as barreiras do recrutamento moderno. Enquanto outras plataformas focam em capturar dados de usuários em formulários intermináveis, nós focamos na **conexão instantânea**. 

Acreditamos que uma vaga de emprego é uma conversa, e toda conversa deve ser fluida, transparente e inteligente. Nosso compromisso é com a **velocidade do negócio** e a **experiência do talento**.

---

## 🛡️ Pilares Tecnológicos (O Coração da Plataforma)

### ⚡ Zero-Account Experience (ZAX)
**O quê:** Uma arquitetura proprietária que elimina a necessidade de contas tradicionais.  
**Por quê:** Reduz o atrito de entrada em 95%. Candidatos não precisam de login para favoritar, e recrutadores gerenciam tudo via tokens de sessão.  
**Quando:** Implementado desde a v0 como pilar central de segurança e UX.

### 🧠 Session Intelligence
**O quê:** Um sistema de inteligência contextual que monitora a jornada do usuário.  
**Por quê:** Para fornecer assistência proativa. Se o recrutador está com dificuldade em descrever benefícios, a IA intervém com sugestões baseadas em tendências de mercado.  
**Quando:** Evoluído na v2 com a integração do Gemini 1.5 Flash.

### 🎨 Design Social Tech (High-End Social Media)
**O quê:** Interface baseada em micro-interações, Glassmorphism e layouts modulares.  
**Por quê:** Para tornar o processo de busca de emprego menos "administrativo" e mais "vivo" e envolvente, como uma rede social premium.  
**Quando:** Consolidado na v3 com o redesign completo da interface.

## 🧠 Deep Dive: O Motor Trampo IA

O **Trampo IA** não é um chatbot; é um motor de execução estruturada que transforma a experiência de quem contrata.

*   **⚡ Feedback em Milissegundos:** Utilizando o modelo Flash do Gemini, analisamos o input do usuário através de um sistema de debouncing que garante que a UI nunca trave.
*   **📊 Recruiter Scoring:** O motor avalia o anúncio sob 4 perspectivas: Clareza, Atratividade, Estrutura e Inclusividade.
*   **🔗 JSON-First Architecture:** Toda a resposta da IA é tratada como um objeto de dados, permitindo que o React renderize feedbacks visuais (checklists, alertas, badges) de forma nativa.

## 💳 Ecossistema de Crescimento (Monetização)

O Trampo Fácil utiliza o **Stripe** para oferecer um modelo de **Boost de Visibilidade** escalável:

1.  **Gema Premium (Diamond UI):** Destaca a vaga no topo do feed com um brilho visual único, aumentando o alcance orgânico.
2.  **Fogo Dourado (Urgent Tag):** Sinaliza urgência crítica, atraindo talentos prontos para contratação imediata.
3.  **Checkout Transparente:** Integração via **React Portals**, garantindo que o pagamento ocorra dentro da fluidez da plataforma.

## ⏳ A Grande Jornada: Histórico de Evolução

### 🌱 v0 — A Gênese (Fevereiro 2026)
*   **O quê:** Fundação técnica e arquitetura de dados.
*   **Por quê:** Para provar que uma plataforma "sem login" era segura e viável.
*   **Impacto:** Estabeleceu a stack React + Vite + Supabase como padrão de alta performance.

### 🏗️ v1 — Base Estrutural e Segurança
*   **O quê:** Implementação do sistema de **Link-Tokens** e busca em tempo real.
*   **Por quê:** Para permitir que recrutadores editassem vagas sem senhas, usando apenas um link seguro enviado por e-mail/sessão.
*   **Impacto:** Validou o conceito ZAX no mercado real.

### 🧠 v2 — O Despertar da Inteligência
*   **O quê:** Integração oficial com **Google Gemini 1.5 Flash**.
*   **Por quê:** Para resolver o problema de "vagas mal escritas" através de consultoria automática via IA.
*   **Impacto:** O Score de Performance tornou-se o recurso mais amado pelos recrutadores.

### 💎 v3 — A Estética Tech-Premium
*   **O quê:** Redesign completo para **Glassmorphism** e lançamento do **Command Hub**.
*   **Por quê:** Para elevar o Trampo Fácil ao nível das grandes redes sociais globais.
*   **Impacto:** Aumento de 40% no tempo de permanência na plataforma devido à interface envolvente.

### 🛡️ v4 — Compliance e Maturidade
*   **O quê:** Central jurídica modular (Privacidade, Segurança, Termos) e FAQ inteligente.
*   **Por quê:** Preparar a plataforma para uso em nível corporativo/enterprise.
*   **Impacto:** Transmissão de confiança institucional para grandes empresas.

### 🚀 v5 — O Salto para a Monetização
*   **v5.0:** Lançamento da Landing Page `/empresas`, focada em conversão B2B.
*   **v5.1:** Lançamento dos filtros "Em Alta" e tags dinâmicas de urgência.
*   **v5.2 (Atual):** **Integração Real Stripe**. Implementação da iconografia de Gema e Fogo Dourado, e o novo **Payment Modal v2**, consolidando a plataforma como um produto completo de ponta a ponta.

## 📦 Guia de Setup Local

1.  **Clonar & Instalar:**
    ```bash
    git clone https://github.com/lukaslimna1/TrampoFacil.git
    npm install
    ```
2.  **Configurar Segredos (.env):**
    ```env
    VITE_SUPABASE_URL=...
    VITE_SUPABASE_ANON_KEY=...
    VITE_STRIPE_PUBLIC_KEY=...
    VITE_GEMINI_API_KEY=...
    ```
3.  **Lançar Dev Server:**
    ```bash
    npm run dev
    ```
