# 🚀 Trampo Fácil

### Recrutamento com Inteligência Humana e Performance Tecnológica.

O **Trampo Fácil** não é apenas uma plataforma de vagas. É um ecossistema inteligente desenhado para eliminar o "buraco negro" dos currículos, conectando talentos e empresas de forma direta, gratuita e humana, potencializado pelo motor proprietário **Trampo IA**.

![Trampo Fácil Banner](src/assets/hero.png)

---

## ✨ Diferenciais da Plataforma

- **100% Grátis & Aberto**: Sem assinaturas premium para ver vagas ou enviar currículos.
- **Conexão Direta**: Candidatura simplificada via WhatsApp, E-mail ou formulários diretos.
- **Foco em Diversidade**: Tags integradas de ações afirmativas e acessibilidade (PcD).
- **Design Premium**: Interface ultra-moderna focada na experiência do usuário (UX).

---

## 🧠 Trampo IA Engine (v3.0)

O coração do Trampo Fácil é o **Trampo IA**, um módulo de inteligência de mercado que analisa dados em tempo real para otimizar o ecossistema:

### Para Recrutadores
- **Score de Performance**: Avalia a atratividade da vaga de 0 a 100%.
- **Análise de Benchmark**: Compara salários e benefícios com a média do mercado (Tech, Design, Gestão).
- **Checklist de Otimização**: Sugestões inteligentes para aumentar o alcance da vaga em até 40%.

### Para Candidatos
- **Match de Valor**: Identifica por que uma vaga é uma boa oportunidade (Salário acima da média, benefícios robustos, etc).
- **Tags "Em Alta"**: Algoritmo que detecta vagas com alta taxa de engajamento e visualizações.

## 🛠️ Funcionalidades Principais

### 🔍 Busca & Descoberta
- Filtros dinâmicos por cargo, empresa e localização.
- **Stories de Filtragem**: Atalhos rápidos para vagas Remotas, PcD e "Em Alta".
- Visualização em Split-Screen (Desktop) para análise rápida de detalhes.

### 📝 Gestão de Vagas
- Publicação simplificada com sistema de tokens para edição posterior.
- Dashboard de insights (visualizações em tempo real).
- Sistema de benefícios modular com ícones inteligentes.

## 💻 Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema Web:

- **Core**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Linguagem**: JavaScript (ES6+)
- **Estilização**: CSS Moderno (Custom Properties, Flexbox/Grid, Glassmorphism)

## ⚙️ Como Executar o Projeto

### Pré-requisitos
- [Node.js](https://nodejs.org/) (versão 18 ou superior)
- NPM ou Yarn

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/lukaslimna1/TrampoFacil.git
   ```

2. Entre no diretório:
   ```bash
   cd TrampoFacil
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. Abra o navegador em `http://localhost:5173`.
